# PROJET DE FIN D'ETUDE

This is a web application developed using **PHP**, **CSS**, and **JavaScript** as part of a final year project (PFE). The system provides a platform for managing products, user accounts, and a shopping cart experience.

---

## üöÄ Features

* **User Management:** Account creation and login functionality (`account.php`).
* **Product Catalog:** Browse products and view details (`aboutus.php`, `addpro.php`).
* **Shopping Cart:** Add items to a cart and manage purchases (`cart.php`, `cartadd.php`).
* **Admin Dashboard:** Dedicated administrative interface to manage products and site content (`admin.php`, `addpro-admin.php`).
* **Feedback System:** Users can leave comments or reviews on the platform (`commentaire.php`).

To run this project locally, you will need a local server environment like **XAMPP**, **WAMP**, or **MAMP**.

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/HajarBoulmane/HK.git]
     ```
2.  **Move to Web Directory:**
    Place the project folder inside your server's root directory (e.g., `C:/xampp/htdocs/`).
3.  **Database Setup:**
    * Open **phpMyAdmin**.
    * Create a new database (check your connection file for the exact name).
    * Import the `.sql` file (if available in the repository).
4.  **Run the App:**
    Open your browser and navigate to `http://localhost/project-folder-name/`.
